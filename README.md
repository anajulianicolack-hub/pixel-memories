# Pixel & Memories

> Um museu digital dedicado a preservar a história e a memória dos jogos clássicos.

---

## Sobre o Projeto

O **Pixel & Memories** é um projeto acadêmico desenvolvido para a disciplina de Desenvolvimento Web, com o objetivo de reunir e aplicar todos os conhecimentos adquiridos ao longo do semestre.

A proposta do site é documentar a história de jogos antigos, apresentando curiosidades, contexto histórico e detalhes que marcaram a indústria dos videogames.

A escolha do tema vem de uma paixão pessoal pelo universo dos retrogames. Mesmo com limitações gráficas e técnicas, os jogos clássicos carregam histórias marcantes, trilhas sonoras inesquecíveis e experiências que continuam vivas na memória de milhões de pessoas.

Este projeto nasce da ideia de que a história desses jogos não deve ser esquecida — e que cada pixel carrega uma memória.

---

## Funcionalidades

- Histórias dos jogos clássicos    
- Easter eggs e fatos interessantes  
- Sistema de cadastro de usuários  
- Carrossel interativo em JavaScript  
- Organização por plataformas clássicas  
- Menu accordion para navegação por console
